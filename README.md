Sales Analytics Dashboard

Project Overview

This Power BI dashboard was developed to analyze sales performance, profitability, customer segments, shipping methods, and operational costs. The dashboard provides interactive insights that help support business decision-making through data visualization and KPI tracking.

Tools Used

* Power BI
* Power Query
* DAX
* Excel

Key Performance Indicators (KPIs)

* Total Orders
* Total Quantity Sold
* Total Sales
* Total Profit
* Total Cost
* Total Shipping Cost

Dashboard Features

* Interactive Filters and Slicers
* KPI Cards for business metrics
* Sales Trend Analysis
* Customer Segment Analysis
* Shipping Mode Analysis
* Profitability Analysis
* City-Level Performance Analysis
* Top Customer Analysis

DAX Measures Used

* SUM()
* CALCULATE()
* COUNT()
* DISTINCTCOUNT()
* DIVIDE()

Insights Provided

* Monitor overall sales performance
* Compare profitability across different segments
* Analyze shipping costs and shipping methods
* Identify top-performing customers
* Evaluate sales distribution across cities
* Track business KPIs in real time

Skills Demonstrated

* Data Cleaning
* Data Transformation
* Data Modeling
* DAX Calculations
* Data Visualization
* Dashboard Development
* Business Intelligence
* Analytical Thinking
